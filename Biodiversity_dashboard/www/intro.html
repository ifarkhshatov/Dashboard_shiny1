With this app, you can visualize selected species observations on the map and see how often each species is observed.

There are several filters available to help you find the data you're looking for:

  • You can filter by scientific name and vernacular name.
  • You can open the advanced options and filter by taxonomic rank, kingdom, and family.
  • If you click on a marker on the map, you can open a detailed info window about a particular observation.

We hope you find this app useful and informative. Thank you for using it!
